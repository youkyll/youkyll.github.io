グーグルドライブから直リンクを生成するブックマークレット

グーグルドライブから直リンクを生成するブックマークレットを作成しました！
画像やソースコード、静的なwebサイトを作成するときに便利なんじゃないでしょうか！

https://gist.github.com/youkyll/fa12b23ee3593c1ce39b

作成にはこちらのサイトを参考にさせていただきました。

Googleドライブから外部参照用アドレスをすばやく取得するブックマークレット

ブックマークレットとは

簡単に言うと、javascriptを利用して今表示しているサイトをより便利に加工しちゃう機能を
ブックマークに保存したもののことを言います。

詳しくはこちらを参照してください！
http://www.lifehacker.jp/2013/04/130402bookmarklet_matome.html

使う前の準備

まずは上記のURLからソースコードをコピーしてください。

その後、適当なページでCtrl Dまたはcmd Dでブックマークを作成しましょう。

その作成したブックマークのURLのところに先ほどコピーしたソースコードを
ぺっと貼ってしまいましょう。
これで前準備は完了です！

使い方

1.フォルダーを一般公開する

まずはグーグルドライブを開き、下記のような画面を表示しましょう。

ここで、直リンクを貼りたいフォルダーを右クリックし、共有ボタンを押しましょう。
詳細設定からアクセスできるユーザーを一般公開に設定します。

2.公開したい画像を置く。

次に公開したい画像をそのフォルダーの中に置きましょう。
このフォルダーにおいたものはURLでだれでもアクセスできてしまうことに注意！

3.ブックマークレットを実行

下記のような画面になったら、先ほど保存していたブックマークレットを実行！！

すると、各ブロックの上にopenというボタンが表示されているのがわかりますか？
そのボタンを押すと、そのままファイルにアクセスすることができます！！

ちなみに、デベロッパーツールのコンソールには
ファイル名とファイルIDが表示されるようにしています。

ぱぱっと適当に作ったスクリプトですが、
ぜひ使ってください(^_^)

ブログを移行しました!!

今日からブログを移行させました!
まだドメインの向き先を変更していないので、
すぐにはyoukyll.comから切り替わらないかと思いますが、そのうち変更しますので少々お待ちを～。

postach.ioから

今回移行に至った理由は利用していたpostach.ioというサービスが有料になってしまったためです。

postach.io

このサイトはevernoteのノートブックをそのまま記事にできるという優れものでした。

evernoteに記事を書いて、そのままpublishedタグを付けて同期すれば投稿できるから、
スマホからでも記事がかけるし、同時にevernoteにバックアップもできる!
更に言えば、サイトのデザインもカスタマイズし放題で万々歳＼(^o^)／

しかし、サイトから突然のお知らせが、、、
無料版をなくし、すべてを有料に移行!?
ある程度は出せるつもりだったのですが、月額1000円ほどするので断念しました。
有料になっても特に機能増えてないしね(´・ω・｀)

ということで、静的ジェネレーターのhexoを利用してgithub pagesを利用することにしました!

hexoへ

hexoはnodejs製の静的ジェネレーターツールです。
これを利用すれば、デザインテンプレートと記事から、htmlを吐き出してくれます!

特に便利なのがgithubの設定を_config.ymlに記載しておけば

でgithubpagesにデプロイしてくれるところ!
これは便利!＼(^o^)／

hexoの細かい使い方やgithub pagesへの上げ方は下記のブログとサイトが参考になりました!

  • 所要時間3分!? Github PagesとHEXOで爆速ブログ構築してみよう！
  • チームブログをGitHubとHexoではじめよう！
  • hexo公式サイト

デザインテンプレートも豊富にあるし、簡単に使えるので、ぜひ使ってみてください!

2015-04-15

ヒッチハイクで名古屋まで行ってきました!(前編)

少し時間が経ってしまいましたが、4月4日にヒッチハイクで名古屋に行ってきました!
とっても充実した旅行だったので、これからヒッチハイクをしようとしている方のためにも報告します!

会社の後輩とヒッチハイクがしたい!と意気投合し、2人で土日に行けそうな名古屋へ行くことに。

結果として、7時に開始し、5台乗り継いで18時に名古屋駅に到着しました!＼(^o^)／

以下に詳しくレポートします!

▼7:00 用賀駅スタート

7時前に用賀駅に集合して、7時からスタート!雨が降りそうで少し焦る。。

▼7:20頃 用賀IJに到着

ヒッチハイクの聖地と言われている用賀IJに着。
すると、既にヒッチハイカーが2組、いや3組も!やはり聖地だけあり、たくさんいました。笑

ヒッチハイクするポイントは2箇所。
左折しようと止まっている車を狙うか、IJに入る手前を狙うかでした。
他のヒッチハイカーと場所取り合戦になるよね!笑
そんなこんなで全く捕まえられず9時をまわる。

途中から左折仕様と停止している車に的を絞り、｢名古屋方面｣、｢海老名SA｣と書かれた看板をひたすら見せる!
信号から列の後ろに向かって見せていく作業を何十往復もやった頃、｢海老名ならいいよ｣とついに声をかけてもらいました!

載せてくれたのはこれから静岡でイチゴ狩りに向かい途中の3人親子。
お子さんは小学3年生で、お母さんと今の小学校の話で盛り上がり、海老名につくのはあっという間でした!
今は授業参観ではなく学校公開といって、いつでも学校へ参観に来ていいらしい。。

▼海老名SA着

海老名SAで休憩をし、
出口付近で再びヒッチハイクを開始!次の大きなサービスエリアの｢足柄SA｣と｢静岡方面｣を掲げました。
すると、30分ぐらいでヒッチハイク成功!

今度はバーベキューに行く途中の親子に乗せてもらいました。
お母さんは小学校の先生、お子さんは次から高校1年生でした!
この親子はとても明るくて、メロンパンまでもらってしまいました。優しさにホント感謝です(^^)

▼足柄SA着

続いての足柄SAでは休憩がてらプレミアムカレーパンで男気ジャンケン。
普通に勝ちにいってしまった。。。
まあ、先輩だしおごりたかったんだけどね!
プレミアムなだけあってサクサクで油もとても美味しかったです!
そして海老名と同じ要領でヒッチハイク。今回は新東名に乗りたかったので、｢浜松SA｣、｢静岡SA｣と書きました!
車がつかまらない中、何故か楽しくなってきて途中で変なハイテンションで騒いでました。笑

そしてついに載せていただける車が!
この方は犬の訓練士の方でした。仕事の途中で、ちょうど清水に犬を迎えに行く途中だったそうです。
犬のしつけ方や、訓練犬の日本大会のことまで教えて下さいました!

しかしこれが悲劇が始まりました。。
